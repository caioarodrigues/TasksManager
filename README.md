# 📌 Aplicativo de Gerenciamento de Tarefas

Este é um aplicativo simples de gerenciamento de tarefas desenvolvido em **Kotlin** para dispositivos **Android**. Ele permite criar, listar, editar, excluir e marcar/desmarcar tarefas como concluídas.

## 📋 Funcionalidades
- ✅ Criar uma nova tarefa com título e descrição
- 📜 Listar todas as tarefas criadas
- ✏️ Editar uma tarefa existente
- 🗑️ Excluir uma tarefa
- 🔘 Marcar e desmarcar tarefas como concluídas

## 🛠️ Tecnologias Utilizadas
- **Linguagem**: Kotlin
- **Plataforma**: Android
- **IDE**: Android Studio
- **Armazenamento de Dados**: Nenhum banco de dados utilizado (dados armazenados temporariamente em memória ou via SharedPreferences)

## 🎨 Interface do Usuário
A interface foi projetada utilizando **RecyclerView** para exibir as tarefas e um **Button** para adicionar novas tarefas.

## 🚀 Como Executar o Projeto
1. Clone este repositório:
   ```bash
   git clone https://github.com/caioarodrigues/TasksManager
   ```
2. Abra o projeto no **Android Studio**.
3. Compile e execute em um emulador ou dispositivo Android.
